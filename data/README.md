# Dataset Information

This project uses the IEEE-CIS Fraud Detection dataset from Kaggle.

Dataset Source:
https://www.kaggle.com/competitions/ieee-fraud-detection

Files used:
- train_transaction.csv
- train_identity.csv
- test_transaction.csv
- test_identity.csv
- sample_submission.csv

Raw dataset files are not included in this repository due to size and licensing considerations.

To run the notebooks, download the dataset from Kaggle and place the files inside the `data/` directory.
